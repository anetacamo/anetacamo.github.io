Get Started

HTML and CSS

HTML stands for hypertext markup language. It is a skeleton of elements a page consists of.

CSS stands for cascading stylesheets and is a way to style elements - tags - defined in the html files. Via css you can for example control color, width, height, style the fonts etc..

cascading means that a style applied to a parent element will also apply to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and other text elements within the body will also get the same color (unless you specify something else)!

Getting a code a code editor

To start you will need an program that will make it easy to write and read the code. Visual Studio Code is completely free and one of the most powerful and easy to use programs available.

Download VS Code here

Create a folder on your desktop and open it in the VS Code. Here, create a new file and call it index.html. This will be a first file a web browser will look for when accessing your folder.

Go to your browser and click CTR + o. Find your folder and open index.html. At this point your file might be empty. But this is a plce you will/can see all the code being executed! You can refresh the page with CTRL + r.

Recommended order of files and folders

Your website might have more then a homepage named index.html. If so, those pages should be placed in the same folder and saved with their specific names: about.html and contact.html You will probably need a folder with images and a folder with styles. index.html structure

Each .html file consist of body and a head.

Head is where you define setting for your page: where you link the styles, give a page name and an icon that will be displayed in the browser window (favicon), etc.

Body is a place where you would write the actual code to be rendered directly 

      <!DOCTYPE.html> 
      <html>
        <head>
          <title>The Homepage Title displayed in the browser windows<title>
          <link rel="stylesheet" href="styles/style.css" />
          <link rel="stylesheet" href="styles/typography.css" />
        </head>
        <body>
          this is where the coding happens!
        </body>
      <html>

Most commonly used HTML tags

div and its family tree

<div></div>

A div itself has no styling, it is a basic unit of the html document. Unstyled and empty, it is see-through and has 0 width and 0 height. It adjusts its size to its content.

pair vs non-pair element

Div is always a pair element. It means it will always consist of a double tag: an opening tag without a dash <div> and a closing tag with a dash</div>. It is an essential feature for wrapping more content in.

A non-pair element. For example <img /> or <input /> They are already content themselfs, they cannot contain anymore elements, therefore they are single tags.

Divs are pretty much all about hierarchy:
